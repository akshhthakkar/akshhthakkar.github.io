An Enterprise Resource Planning (ERP) system built using React.js for the frontend, Node.js and Express.js for the backend, and MongoDB as the database, exposed via a REST API. It also includes a dedicated admin panel for comprehensive control and management.

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- REST API
