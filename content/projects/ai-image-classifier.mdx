This AI Image Classifier is a user-friendly web application built with Streamlit that lets you upload images and classify them using the pre-trained MobileNetV2 model from TensorFlow. It identifies objects in images and displays the top-3 predictions with confidence scores. Perfect for exploring AI-powered image recognition!

## Tech Stack

- **Python**
- **TensorFlow**
- **OpenCV**
- **MobileNetV2**
- **Streamlit**
