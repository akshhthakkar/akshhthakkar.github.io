Create New Blog Post

To add a new blog post:

  1. Create a new HTML file inside the blog/ folder. For example: blog/my-new-post.html
  2. Use this basic blog template (copy from an existing blog post):
    3. <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>My New Blog Post</title>
</head>
<body>
  <h1>My New Blog Post</h1>
  <p>Write your blog content here.</p>
</body>
</html>
  3. Then, open blog.html and add a new entry in the blogPosts array:
    4. { title: "My New Blog Post", url: "blog/my-new-post.html", date: "2025-05-25" }
  4. Save and push your changes to GitHub. Your blog page will now show the new post!
